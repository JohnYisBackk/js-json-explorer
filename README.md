# JSON Explorer 

An interactive JSON utility built with vanilla JavaScript that allows users to validate, format, minify and visualize JSON data in a clean and modern interface.

## ✨ Features

- ✅ Validate JSON input
- 🎨 Format JSON with proper indentation
- ⚡ Minify JSON
- 🌳 Interactive JSON tree visualization
- 📊 Node counter
- 📋 Copy formatted output
- 🧹 Clear input and reset the application
- 📏 Live input/output character counter
- 🚨 Error handling with status messages
- ⌨️ Ctrl + Enter shortcut for quick formatting

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript (ES6)
